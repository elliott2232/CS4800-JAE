This file credits and describes the various softwares & tools used to implement our project thus far. These tools do not need to be stored in this file to run the project, but for documentation's sake we've included the following information.



MongoDB: This service provided the servers for our database. Entries are stored as dictionaries allowing easy handling of article data.

Pymongo: This python package allowed us to interact with our Mongo database within a script.

Flask: This web framework allowed us to run our python scripts within our site.
